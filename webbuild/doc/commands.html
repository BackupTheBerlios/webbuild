Commands

Overview

WebBuild currently supports the following commands:

  • make <yourpage>.html
  • make <yourpage>.bak
  • make drophtml

make <yourpage>.html

This command creates a HTML page called yourpage.html. Please note, that make will fail if the appropriate source-file is missing. You cannot rebuild the page if you did not changed the source file after last build.

make <yourpage>.bak

This command creates a backup file of yourpage.html and deletes the original so that you can execute make <yourpage>.html again, while the source file has not changed.

make drophtml

You should use this command very carefully because it deletes all HTML files in the current directory. This includes also "template.html".
